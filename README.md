# Sistema Interno de Gestão de Telefonia VoIP

Sistema corporativo desenvolvido para gerenciamento da infraestrutura de telefonia VoIP em ambiente organizacional.

A aplicação permite o controle, consulta e rastreamento de dispositivos, incluindo histórico de entregas, geração de termos de responsabilidade e organização hierárquica por Secretaria, Departamento, Setor e Pessoa.

## 🎯 Principais características

- Gestão de telefones e ramais
- Controle de entregas e responsáveis
- Histórico completo de movimentações
- Geração de termos em PDF
- Organização hierárquica dos dados
- Agrupamento lógico por troncos

## 🏗️ Arquitetura

O sistema foi desenvolvido utilizando o padrão MVC com Flask, garantindo separação de responsabilidades e escalabilidade.

## 🛠️ Tecnologias

- Python
- Flask
- SQLAlchemy
- MySQL (PyMySQL)
- ReportLab
