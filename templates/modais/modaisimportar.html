Importação de Telefones

Selecione o arquivo Excel (.xlsx) contendo os dados dos telefones para realizar a importação.

Apenas arquivos .xlsx são suportados.
Atenção!
Esta operação irá alterar todo o banco de dados de telefones e não poderá ser desfeita após a confirmação.

O modelo de planilha Excel para importação deve ser solicitado ao Centro de Informática.
